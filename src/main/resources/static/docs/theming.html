Theming

Easy theming and component changes with Sass variables.

Introduction

Looper comes with scss directory in src/assets. In the scss directory you will look file structure that looks like this: 

scss/
├── bootstrap/
├── components/
├── bootstrap.scss
├── custom.scss
├── main.scss
├── pace.scss
└── quill.scss

You can ignore pace.scss and quill.scss file, as this is a per plugin styles. Whenever possible, avoid modifying Looper core files except variables files, as they are contains global variable. Please use custom.scss instead if you want to modify or creating new components.

Bootstrap

This folder is where custom bootstrap styles lives. With some effort, you can modify the look of custom bootstrap components by touching _variable-overrides.scss.

Components

The components directory contains all new components that included in main.scss. Each components separated in their own files in this directory. You can edit the look of each components by touching _variables.scss.
